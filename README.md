# Task 4: Password Security & Authentication Analysis

This repository contains my Task 4 assignment on **Password Security and Authentication**.

## Topics Covered
- Password storage (Hashing vs Encryption)
- Hash types (MD5, SHA-1, bcrypt)
- Password hash generation
- Dictionary and brute force attacks
- Why weak passwords fail
- Multi-Factor Authentication (MFA)
- Strong authentication recommendations

## Objective
To understand how passwords are stored securely, how weak passwords are attacked, and how strong authentication mechanisms protect systems.
